# AI-MATH-SOLVER

An AI-powered math solver that:
- Solves equations using SymPy
- Uses LLM fallback (Gemini + Groq)
- Provides step-by-step explanations

## Features
- Equation solving
- Word problem solving
- Smart fallback system
- Clean explanations

## Tech Stack
- Python
- SymPy
- Gemini API
- Groq API
